Flexbox y CSS Grid

Las herramientas que cambiaron el diseño web

Flexbox

  • Alineación fácil en 1D
  • Eje principal y secundario
  • Ideal para menús y tarjetas

CSS Grid

  • Diseño en 2 dimensiones
  • Filas y columnas
  • Layouts complejos simples

¿Cuándo usar?

  • Flex → distribución lineal
  • Grid → estructura de página
  • Se pueden combinar
