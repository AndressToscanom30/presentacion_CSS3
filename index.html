¿Qué es CSS3?

CSS3 es la última evolución de las hojas de estilo en cascada, permitiendo crear páginas web modernas, responsivas y visualmente atractivas.

Novedades de CSS3

  • Animaciones y transiciones
  • Flexbox y Grid Layout
  • Sombras y bordes redondeados
  • Media Queries para diseño responsivo
